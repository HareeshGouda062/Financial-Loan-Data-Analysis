# Financial Loan Data Analysis

## Project Overview
This project analyzes financial loan data to identify loan trends, customer behavior, funding distribution, and risk-related insights using Python and data visualization techniques.

## Tools & Technologies
* Python
* Pandas
* NumPy
* Matplotlib
* Plotly
* Jupyter Notebook

## Key Analysis Performed

* Loan amount analysis
* Home ownership analysis
* Monthly application trends
* Interest rate analysis
* Loan status analysis
* Funding distribution visualization
* Treemap visualization using Plotly

## Project Features

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Interactive visualizations
* KPI metrics
* Trend analysis

## Sample Visualizations
* Treemap Charts
* Area Charts
* Line Charts
* Bar Charts

## Business Insights
* Identified high funded loan categories
* Analyzed application growth trends
* Evaluated borrower patterns
* Studied ownership-wise funding behavior

## Conclusion
This project demonstrates practical data analysis and visualization skills using financial domain data.
